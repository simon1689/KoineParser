Introduction

I'm Simon. As a software engineer and as a Greek geek, I have the ability to create tools which help me learn Greek. If I could find the time that is...

I write the start and end date for reading books. On 11 October 2013, I started reading William Mounce's Basics of Biblical Greek. As with everything new you start with, you start very good, but don't end that good. Well, that's exactly what happened to me. I went on and off and didn't even start with the verb system.

But in 2020, around August, I decided that I need to up my game and at least finish the book. Having started with the verb system I released how important it was to recognize the different forms of the words and how important is that for the meaning. I started creating exercises for myself to write out the different paradigms. I would write every paradigm sometimes 6 times a week.

All that to say, that recognizing the forms is very important. From Darryl Burling, I learned that setting goals for Greek is very important. So I set up a goal toward the end of October 2020:

Read, parse, and translate 1-3 John by the end of the year.

For reading and translating there are various tools. I personally use TheWord Bible software. It is a free Bible software which I've used for very long and enjoy. You can buy modules and there a ton of free modules. I can quickly search for words and their meanings and even morphologies. So that's no problem. Parsing is also possible to do with a real Bible, but that's kinda boring to go word-for-word.

From the Daily Dose Of Greek I learned of MasterGreek.com. I very much liked the concept and what I could do. Because I often used the website on my phone, the experience irritated me because when I would scroll down to choose an option, the browser would refresh and I would lose my progress. Furthermore, it did not provide me with the option to achieve my goal: Read, parse, and translate 1-3 John.

Having found some spare time, and sadly not learning but programming Greek instead, I managed to create something which would bridge the gap. I pray that it is also a blessing to you and that it may help you learn Greek better.

How to use

Per default, words which are not declinable are left out. When you don't choose any of the 'types', you get all unique declinable word forms which occur in the chosen range. This also includes types of words which are not included in the 'types' checkboxes on the home page (e.g., conditional particles).

Definitions are also included for words from Dodson's Greek Lexicon as well as Mounce's definitions. In some cases, another lexicon is used which I'm not sure which one it is.

In the case of verbs as a type (on the main page), verbs include participles as well as infinitives (which can be chosen as the 'mood' of the word.

For verbs, you will see an additional checkbox asking if you want secondary tenses. By secondary tenses, I mean 2nd Aorist, 1st and 2nd Perfect. By default, answering with Aorist for a 2nd Aorist verb is acceptable. But when the secondary tenses option is checked, the secondary tenses appear as options on the parsing page and the exact tense has to be given for a word to be accepted as correct.
